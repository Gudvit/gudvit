Notifications

How to create notifications for IOS and Android with ionic 3

Installation

ionic cordova plugin add phonegap-plugin-push npm install --save @ionic-native/push

Add this plugin to your app's module

import { Push } from '@ionic-native/push'

Usage

Add this in app.component.ts page

import { Push, PushObject, PushOptions } from '@ionic-native/push' constructor (private push: Push) { }

It is best to initialize push notifications when the user is logged in

And call it when the platform is ready also in app.component.ts

Now you need to configure for iOS

Push notifications will go through the APNS server for iOS and for this we need to create a certificate

Go to https://developer.apple.com/account/ios

Create a certificate

After that, you need to include a push notification in the current certificate

Create a provision profile

And after that we download the application through xcode and select our provision profile

And check push notifications on IOS devices

Now you need to configure for Android

Push notifications will go through the FCM server for Android and for this we need to create a project in Firebase console

https://console.firebase.google.com
